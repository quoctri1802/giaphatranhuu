# Hướng dẫn thiết lập dự án Gia phả Họ Trần (Đã cập nhật)

Do thư mục có dấu cách nên lệnh khởi tạo tự động bị lỗi. Tôi đã tự tạo các file cấu hình cần thiết (`package.json`, `tsconfig.json`, `next.config.mjs`) cho bạn.

Bây giờ bạn chỉ cần thực hiện các bước sau:

1. Mở Terminal tại thư mục `g:\Gia pha online`.
2. Chạy lệnh cài đặt thư viện:
   ```powershell
   npm install
   ```
3. Khởi tạo Prisma:
   ```powershell
   npx prisma init
   ```

Sau khi hoàn thành, hãy báo cho tôi biết. Tôi sẽ bắt đầu tạo các thư mục và code cho ứng dụng!
